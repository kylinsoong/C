=== 查看 CMP 工作状态

[source, text]
.*1. 查看 CMP 是否开启（0 为开启，1 为关闭）*
----
# tmsh list sys db provision.tmmcount
sys db provision.tmmcount {
    value "0"
}
----

[source, text]
.*2. 查看 CMP 是否开（show vs）*
----
# tmsh show ltm virtual all | grep CMP
  CMP              : enabled   
  CMP Mode         : all-cpus  
----
